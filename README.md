# ChromaBridge (flux-ui-adapter)

Real-time DOM mutation capture and screenshot ingestion platform.

A monorepo containing a **WebSocket ingestion server** (Node.js / Fastify / SQLite) and a **Chrome extension** (Manifest V3) that captures DOM changes and periodic screenshots from any web page, streams them to the server, and provides a live session replay viewer.
